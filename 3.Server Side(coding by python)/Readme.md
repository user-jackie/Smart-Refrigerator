# Python后端代码

接入了阿里云百炼的ASR(自动语音识别)、LLM(大语言模型)、TTS(文本转语音)这3种技术的api，能实现语音助手的功能。

接入了和风天气的api，能够查询某个地区的天气信息

后端代码的使用没有任何限制，只需要搭建好python环境即可使用，一般执行文件时报错`from xxx import aaa`，这种一般使用`pip install xxx`命令下载xxx库即可，但是有些库可能变了名字，导致下载的时候找不到！此代码中库有一个是变了名字的，这种缺库的问题推荐是现在浏览器上找答案，找不到再问ai(你敢问ai真敢忽悠)。

下载好库后直接使用python命令执行代码即可



注意：

1.和风天气的后端代码在什么时间启动都可以，但是阿里云百炼的api必须在前端qt代码启动前启动，写启动脚本的时候需要留意一下顺序。

2.阿里云的需要使用个人的api-key，和风天气需要使用个人的api-key和api-host，创建账号之后再创建工程即可。

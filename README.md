# Smart-Refrigerator
A smart refrigerator based on rk3568

如果学习过相关内容的同学就会发现这个东西是基本没有技术难度的，那为什么要做这个东西呢，因为是学校的课，懂得都懂。虽然没有技术难度，但毕竟是本人第一个伪全栈项目(因为rk3568的高速板用的是开发板)，还是比较有纪念意义的。

说一下感想吧，现在ai的发展真是日新月异。如果让我从头写这些qt代码可能一个月都写不出来，但是现在用ai来写一个星期功能就大差不差，还是用的免费的Claude Code版本。个人感觉学习技术的成本越来越低，特别是和应用型代码相关的技术，也许以后使用一门技术只需要了解这门技术叫什么名字就行了？遵循规则是没有出路的，要么制定规则，要么打破规则。

# 冰箱外壳

文件分为装配体文件和用于打印的文件

1.装配体文件

`装配体.SLDASM`文件是冰箱外壳的总文件。

2.打印文件在名字叫`打印文件`的文件夹中，都是.stl类型文件，由于使用的是自己的打印机，每条边长度不能超过20cm，所以有些模块进行了分段打印，如果自己打印机比较大或者给在淘宝上打印可以不用使用这种文件。
